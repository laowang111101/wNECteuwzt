# 智能停车系统 python506

## 项目概述


## 技术栈

- 开发语言：Python 3.9
- 开发工具：PyCharm
- 后端框架：Django
- 前端框架：Vue.js
- 数据可视化：echarts
- 数据库：MySQL 5.7/8.0

## 功能模块

### 游客模块

- 首页浏览
- 车位信息查询
- 公告信息浏览
- 个人中心

### 管理员模块

- 主页管理
- 车主信息管理
- 车辆信息管理
- 车位类型管理
- 车位信息管理
- 车辆入场管理
- 车辆出场管理
- 系统管理
- 个人中心

## 系统角色

- 游客：查看车位信息，阅读公告等。
- 管理员：对系统进行详细操作，包括车位、车辆及用户信息的维护。

## 运行环境

- 开发环境：PyCharm
- Python版本：3.9
- 数据库版本：MySQL 5.7或8.0
